# Contact Bot

Replace that boring old contact form with your own chatbot. Interface through a React component that shows up as a modal on your React site.

### Installing

```
git clone https://github.com/oandrienko/contact-bot
cd contact-bot
npm install
```

To start the server:

```
npm run start
```
